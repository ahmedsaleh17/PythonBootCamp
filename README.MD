# Python BootCamp

A beginner-friendly collection of Python examples and exercises. This repository covers a variety of core topics and is organized into simple scripts to help learners practice and explore Python concepts.

## Contents

- Basic data structures: lists, tuples, sets, dictionaries
- Functions and error handling
- Web scraping and working with APIs
- Object-oriented programming
- Problem-solving examples in `ProblemSolving/`
- Database and module examples in `python_DE/` and `PythonModule/`

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/ahmedsaleh17/PythonBootCamp.git
   ```
2. Navigate to the folder and run any script using Python 3:
   ```bash
   cd PythonBootCamp
   python 01_list.py
   ```
